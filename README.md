# Deep-Learning-vs-Classical-Machine-Learning-for-Log-Anomaly-Detection-A-SIEM-Style-Evaluation
Comparative study of Deep Learning and Classical Machine Learning for log anomaly detection on the HDFS dataset. Evaluates LSTM, Autoencoder, LSTM-AE, and Random Forest using both traditional metrics and SIEM-oriented measures such as False Positive Rate, Alert Volume, and operational efficiency.
